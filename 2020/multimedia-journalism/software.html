Software and other tools

When you try to install some of the software below on the Mac, you may be warned that it comes from an unidentified developer. If this happens, right-click the installer file and select Open.

Firefox

Some of the tools we will use are Firefox add-ons. Ensure that you have the latest version of this popular web browser. Download Firefox from here.

Open Refine

A powerful tool for data cleaning, once owned by Google, now open source. Download version 3.2 from here. When launched, Open Refine will work in your default web browser; note that your data remains on your computer, and is not uploaded to the web.

Google Sheets

Our spreadsheet application for these classes will be Google Sheets. Use from your Google Drive account.

Plotly

A web app for making charts from data. Sign up for a free account here.

R and R Studio

R is a software environment and programming language for statistical computing and graphics. Download from here. R Studio provides a user interface that makes it much easier to use. Download the latest free version of RStudio Desktop from here.

Sublime Text

For some exercises, you will need a text editor optimized for authoring web pages. Download from here.

ColorBrewer

Your go-to resource for color schemes to encode data. Web app: no installation required.

Datawrapper

Web app developed by German journalism organizations that has been used to make simple online charts and maps by several leading news outlets. No installation required, but you will need to sign up for a free Datawrapper account using the Login/Sign Up link at top right of the home page.

Flourish

Templates for more sophisticated online visualizations including network graphs, animated bar charts, treemaps, and more, from the UK digital design studio Kiln. Sign up for a free account on the home page.
